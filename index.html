List:

HTML has two types of list -ordered and unordered. Each item of your list has to be inclosed in a tag. The syntax of craeting is fairly simple.

Suppose you want to creat a list like:

  • Item 1
  • Item 2
  • Item 3

The following code wiil do easily job for you 

    <ul>
       <l>Item 1</l>
       <l>Item 2</l>
       <l>Item 3</l>
    </ul>

This, by the way was an example on unordered list. For an ordered list, all you need to do replace <ul> with <ol> and </ul> with </ol>.

Let`s see what the following code does.
